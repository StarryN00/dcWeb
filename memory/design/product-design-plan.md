## 一、项目概述

### 1.1 项目定位
打造一个**极简主义风格的装修公司官网**,集品牌展示、案例营销、潜客获取、项目管理于一体的全功能平台。

### 1.2 目标用户
- **C端用户**: 有装修需求的业主(住宅、公寓、别墅、商业空间)
- **B端用户**: 公司内部管理人员(案例管理、潜客跟进)

### 1.3 核心价值
- **用户侧**: 快速了解公司实力 → 浏览真实案例 → 一站式获取装修报价
- **公司侧**: 展示品牌形象 → 自动化潜客收集 → 智能评分分级 → 高效项目管理

### 4.2 潜客评分算法

```javascript
/**
 * 潜客评分系统
 * 总分范围: 50-100分
 */

function calculateLeadScore(lead) {
  let score = 50; // 基础分

  // 1. 预算评分 (最高+30分)
  if (lead.budget >= 50) score += 30;
  else if (lead.budget >= 30) score += 25;
  else if (lead.budget >= 20) score += 20;
  else if (lead.budget >= 10) score += 15;
  else score += 10;

  // 2. 面积评分 (最高+10分)
  if (lead.area >= 200) score += 10;
  else if (lead.area >= 150) score += 8;
  else if (lead.area >= 100) score += 6;
  else if (lead.area >= 80) score += 4;
  else score += 2;

  // 3. 时间评分 (最高+10分)
  switch(lead.timeline) {
    case '1个月内': score += 10; break;
    case '1-3个月': score += 7; break;
    case '3-6个月': score += 5; break;
    case '6个月以上': score += 3; break;
    case '暂无计划': score += 0; break;
  }

  return Math.min(score, 100); // 最高100分
}
```

**评分级别划分**:
- **A级潜客** (90-100分): 高预算+大面积+时间紧急 → 优先跟进
- **B级潜客** (75-89分): 中高价值客户 → 重点关注
- **C级潜客** (60-74分): 中等价值客户 → 常规跟进
- **D级潜客** (50-59分): 低价值客户 → 标准流程

### 4.3 案例筛选逻辑

```javascript
/**
 * 案例筛选函数
 */
function filterCases(cases, filters) {
  return cases.filter(caseItem => {
    // 1. 状态筛选: 只显示已发布案例
    if (caseItem.status !== 'published') return false;

    // 2. 风格筛选
    if (filters.style && caseItem.style !== filters.style) {
      return false;
    }

    // 3. 面积范围筛选
    if (filters.minArea && caseItem.area < filters.minArea) {
      return false;
    }
    if (filters.maxArea && caseItem.area > filters.maxArea) {
      return false;
    }

    // 4. 价格范围筛选
    if (filters.minPrice && caseItem.price < filters.minPrice) {
      return false;
    }
    if (filters.maxPrice && caseItem.price > filters.maxPrice) {
      return false;
    }

    return true;
  });
}
```

## 五、数据模型设计

### 5.1 案例数据模型 (Cases)

```typescript
interface Case {
  id: string;                    // 唯一标识
  title: string;                 // 案例标题
  location: string;              // 项目位置
  style: CaseStyle;              // 装修风格
  area: number;                  // 面积(㎡)
  duration: number;              // 工期(天)
  price: number;                 // 价格(万元)
  images: string[];              // 图片URL数组
  description: string;           // 项目描述
  testimonial: string;           // 业主感言
  foreman: {                     // 施工队长
    name: string;
    phone: string;
  };
  stage: string;                 // 装修阶段
  featured: boolean;             // 是否推荐
  status: 'published' | 'draft'; // 发布状态
  createdAt: Date;               // 创建时间
  updatedAt: Date;               // 更新时间
}

type CaseStyle =
  | '现代'
  | '北欧'
  | '工业'
  | '侘寂'
  | '轻奢'
  | '极简'
  | '中式'
  | '欧式';
```

### 5.2 潜客数据模型 (Leads)

```typescript
interface Lead {
  id: string;                    // 唯一标识
  name: string;                  // 姓名
  phone: string;                 // 手机号(11位)
  propertyType: PropertyType;    // 物业类型
  area: number;                  // 面积(㎡)
  budget: number;                // 预算(万元)
  styles: CaseStyle[];           // 风格偏好(多选)
  stage: RenovationStage;        // 装修阶段
  timeline: Timeline;            // 时间规划
  score: number;                 // 自动评分(50-100)
  status: LeadStatus;            // 跟进状态
  submittedAt: Date;             // 提交时间
  updatedAt: Date;               // 更新时间
}

type PropertyType =
  | '住宅'
  | '公寓'
  | '别墅'
  | '商业空间'
  | '其他';

type RenovationStage =
  | '仅设计'
  | '设计+施工'
  | '仅施工'
  | '仅监理';

type Timeline =
  | '1个月内'
  | '1-3个月'
  | '3-6个月'
  | '6个月以上'
  | '暂无计划';

type LeadStatus =
  | '待跟进'
  | '已联系'
  | '已预约'
  | '已成交'
  | '已放弃';
```

## 十、风险与挑战

### 10.1 技术风险
- **图片加载性能**: 采用懒加载 + WebP格式 + CDN
- **表单验证复杂度**: 使用成熟的表单库(React Hook Form)
- **移动端体验**: 移动优先设计 + 充分测试

### 10.2 业务风险
- **潜客评分准确性**: 需要根据实际数据调整算法权重
- **垃圾信息过滤**: 添加手机号验证 + 图形验证码

### 10.3 维护风险
- **内容更新频率**: 提供简单易用的后台管理界面
- **系统扩展性**: 预留接口,支持未来新功能
