# 半包装修指导工具 - 产品规划

## 📋 项目概述

**产品定位**: 独立的半包装修指导工具（轻量化Web应用）

**核心价值**: 帮助首次装修业主清晰了解8个装修阶段的流程、采购清单和注意事项，减少装修焦虑

**技术特点**: 纯前端实现（HTML/CSS/JS + Tailwind），参照 `generated-page.html` 的极简设计风格

---

## 🎯 产品定位

### 目标用户
- **主要用户**: 首次装修的年轻家庭（28-40岁），选择半包模式
- **次要用户**: 装修公司客服/设计师（作为客户教育工具）

### 核心痛点与解决方案
| 痛点 | 解决方案 |
|------|---------|
| 信息混乱，PDF/微信记录散落 | 结构化知识库 + 可视化时间轴 |
| 不知道什么时候该买什么材料 | 交互式检查清单 + 时间节点提醒 |
| 与装修公司交底困难 | 标准化术语 + 可导出清单 |
| 进度不清晰，焦虑感强 | 进度可视化 + 完成度百分比 |

### 价值主张
**"你的半包装修智能助手 - 让每个阶段清晰可控"**
- ⏱️ 省时间: 不用反复查资料
- 💰 省金钱: 避免时间节点错误导致返工
- 😌 省心力: 结构化呈现，减少焦虑

---

## 🏗️ 功能架构

### 核心设计理念
**"一个中心，四个视角"** - 同一份装修数据，四种查看方式

```
              [装修数据核心]
            8个阶段 × 采购事项
                    |
    ┌───────┬───────┼───────┬───────┐
    │       │       │       │       │
 [时间轴] [清单] [问答] [知识库]
 全局视角 执行视角 引导视角 查询视角
```

### 四大功能模块

#### 1. 时间轴视图（全局视角）
- **作用**: 可视化展示8个装修阶段的全貌
- **特点**:
  - 垂直时间轴设计（参考 generated-page.html 风格）
  - 当前阶段高亮 + 呼吸灯动画
  - 点击展开阶段详情
  - 显示每阶段耗时、完成度

#### 2. 交互式检查清单（执行视角）
- **作用**: 提供可操作的待办事项
- **特点**:
  - 按优先级排序（紧急/重要/常规）
  - 可勾选 + LocalStorage自动保存
  - 进度百分比显示
  - 导出PDF功能（V2）

#### 3. 问答式引导（引导视角）- V2功能
- **作用**: 通过对话帮助小白用户快速定位需求
- **特点**:
  - 5-8个问题快速筛选
  - 生成个性化待办清单
  - 基于用户情况给建议

#### 4. 知识库/FAQ（查询视角）
- **作用**: 快速检索和深度阅读
- **特点**:
  - 关键词搜索（使用 Fuse.js）
  - 按阶段分类浏览
  - 热门问题快速入口
  - 关联推荐

## 🎨 设计规范

### 视觉设计（参照 generated-page.html）
- **色彩系统**: Stone色系（#1C1917, #78716C, #F5F5F4等）
- **字体**: Inter，16px基准字号
- **圆角**: 0.5rem (按钮), 0.75rem (卡片), 1rem (大卡片)
- **阴影**: 轻柔的 Tailwind 阴影系统

### 交互设计原则
1. **渐进式披露**: 默认简洁，悬停/点击展开详情
2. **即时反馈**: 勾选立即保存 + 动画反馈
3. **一致性**: 图标/色彩/布局统一
4. **容错性**: 重置进度二次确认 + 撤销功能

### 响应式设计
- **移动端** (< 640px): 底部Tab导航，单列布局
- **平板** (768px+): 顶部导航，双列网格
- **桌面** (1024px+): 侧边栏 + 主内容区

   - 按阶段分类浏览

4. **首页导航**
   - 当前进度概览
   - 快速入口切换

5. **基础交互**
   - 响应式设计
   - 流畅动画效果

- 问答式引导（5-8个问题）

## ✅ 验证方式

### 功能验证
1. **时间轴**: 检查8个阶段是否正确显示，点击展开是否正常
2. **清单**: 勾选待办事项后刷新页面，检查状态是否保存
3. **搜索**: 搜索"地漏"、"壁挂马桶"等关键词，验证搜索准确性
4. **进度**: 勾选不同数量事项，检查进度百分比是否正确计算

### 体验验证
1. **移动端**: 在手机浏览器打开，检查布局和交互
2. **加载速度**: 使用 Chrome DevTools 检查首次加载时间 < 3秒
3. **离线使用**: 断网后刷新，检查 LocalStorage 数据是否可用

### 内容验证
1. **完整性**: 对照原始 MD 文件，确认所有阶段和事项已包含
2. **准确性**: 抽查10个关键事项（如"壁挂马桶"），验证描述准确
3. **时间节点**: 检查"提前X天"等时间描述是否正确

## 💡 核心优势总结

| 维度 | 传统方式 | 本产品 |
|------|---------|-------|
| 内容组织 | PDF文档，线性阅读 | 多视图，按需查看 |
| 进度管理 | 纸质清单或Excel | 交互式，自动保存 |
| 可访问性 | 需下载文件 | 打开即用 |
| 查询效率 | Ctrl+F | 智能搜索 + 关联 |
| 学习曲线 | 需通读全文 | 渐进式披露 |

**一句话总结**: 将散落的装修知识转化为结构化、可交互的智能工具，让半包装修从"不知所措"变成"清晰可控"。
